# /writing-style

Load the Zinsser writing style guidance into context.

## When to Use

**Invoke this skill BEFORE any task that involves writing or editing text:**
- Creating blog posts
- Generating social/distribution content
- Writing documentation
- Editing existing content
- Any task that produces written output

## Working Directory

Content Conductor launches Claude from the project directory (`projects/<id>`). Paths below are relative to that directory.

## Action

Read `references/writing-style.md` and apply this guidance to all writing that follows.
