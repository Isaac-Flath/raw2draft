---
name: blog-schedule
description: Review, plan, or update the publishing schedule for blog posts in the content repository.
---

Use this skill for blog scheduling.

Workflow:

1. Inspect post metadata and dates using repo scripts or direct file reads.
2. Identify drafts, scheduled posts, and publish-ready posts.
3. Recommend date changes only when they support a clear publishing plan.
4. Preserve existing frontmatter conventions.

Be explicit about exact dates.
