description: Create or prepare a Raw2Draft content project folder with the expected source, content, social, image, screenshot, and video directories.
---

Use this skill to initialize a content project.

Workflow:

1. Confirm the target project slug or infer one from the title.
2. Create the project directory if needed.
3. Ensure these directories exist: `source/`, `content/`, `social/`, `images/`, `screenshots/`, and `video/`.
4. Add a brief project note only if it helps the workflow.

Do not overwrite existing project files.
