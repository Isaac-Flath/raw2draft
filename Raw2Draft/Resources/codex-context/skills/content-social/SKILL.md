Use this skill to create social distribution copy.

Workflow:

1. Read the source blog, transcript, or project notes.
2. Extract the strongest claims, examples, and contrarian points.
3. Create platform-appropriate variants in `social/` when working in a project.
4. Keep posts specific and avoid generic engagement bait.

Prefer a few strong variants over many weak ones.
