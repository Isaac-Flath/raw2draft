name: content-status
description: Inspect the current content workspace and summarize project status, missing assets, generated outputs, and likely next steps.
---

Use this skill to answer "what is the status?" for a content project.

Workflow:

1. Identify the workspace root and project directory.
2. Inspect `source/`, `content/`, `social/`, `images/`, `screenshots/`, and `video/` when present.
3. Check for source files, transcripts, blog drafts, social drafts, generated media, and publish-ready outputs.
4. Summarize what exists, what is missing, and the next practical action.

Keep the response short and operational.
