---
name: content-transcribe
description: Transcribe or prepare transcription work for source audio/video files in a Raw2Draft content project.
---

Use this skill when a project has source audio or video that needs text.

Workflow:

1. Inspect the project's `source/` directory for audio or video.
2. Prefer existing repo scripts or `just` recipes for transcription.
3. If no script exists, explain the exact command or tool you would use.
4. Write or place transcript output under `content/` using the repo's existing naming conventions.

Do not invent transcript text.
