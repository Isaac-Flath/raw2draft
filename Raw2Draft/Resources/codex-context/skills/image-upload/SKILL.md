---
name: image-upload
description: Prepare or run the repo's local image upload workflow for content assets.
---

Use this skill when uploading images for a blog or content project.

Workflow:

1. Identify the image files and target post/project.
2. Inspect existing upload scripts or `just` recipes.
3. Validate filenames, dimensions, and destination paths before upload.
4. Return final URLs or paths when available.
